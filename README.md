# README
* Ruby 2.4.1

* Rails 5.1.4

* System dependencies

run:
`bundle install`

* Configuration
change `database.yml` if you need

* Database creation:

run:
`rake db:create`

* Database initialization

run:
`rake db:migrate`
`rake db:seed`

* How to run the test suite

`rails s`

The server start in default port:

localhost:3000

* Test

run:
`rspec spec`
